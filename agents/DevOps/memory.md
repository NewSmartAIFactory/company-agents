# DevOps Engineer Memory

## Memory Scopes

- Environment settings
- Deployment decisions
- Incidents
- Runbooks

## Memory Write Rules

Write memory when:

- A decision is approved
- A requirement changes
- A technical convention is established
- A recurring bug pattern is found
- A project risk changes status
