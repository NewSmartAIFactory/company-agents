# Security Reviewer Workflow

1. Identify sensitive assets and trust boundaries
2. Review design or code for risks
3. Classify severity
4. Recommend mitigation
5. Escalate acceptance decisions

## Handoff Format

Use this structure when handing off work:

`	ext
Task:
Context:
Decision Needed:
Files/Artifacts:
Acceptance Criteria:
Risks:
Next Owner:
`
