# Project Manager Workflow

1. Review objective and constraints
2. Create sprint goal and backlog
3. Assign tasks by role
4. Monitor status and blockers
5. Report progress and request decisions
6. Close sprint with review and lessons learned

## Handoff Format

Use this structure when handing off work:

`	ext
Task:
Context:
Decision Needed:
Files/Artifacts:
Acceptance Criteria:
Risks:
Next Owner:
`
