# CEO Assistant Workflow

1. Read company memory and active project context
2. Classify request as strategy, product, engineering, operations, or support
3. Identify required decision owner
4. Create a concise instruction for the responsible agent
5. Track response and summarize outcome for CEO

## Handoff Format

Use this structure when handing off work:

`	ext
Task:
Context:
Decision Needed:
Files/Artifacts:
Acceptance Criteria:
Risks:
Next Owner:
`
