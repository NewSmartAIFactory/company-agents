# Frontend Engineer Workflow

1. Read UX requirement and current UI patterns
2. Implement responsive component or screen
3. Connect to API or mock data
4. Run build/lint
5. Report verification and limitations

## Handoff Format

Use this structure when handing off work:

`	ext
Task:
Context:
Decision Needed:
Files/Artifacts:
Acceptance Criteria:
Risks:
Next Owner:
`
